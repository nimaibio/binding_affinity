ML-VSPred

Machine Learning Based Models for Virtual Screening

Virtual Screening Workflow for ML-VSPred

Please select "Use NaturePro" for virtual screening:

  • Yes: The system will screen natural compounds against therapeutic proteins.
  • No: Upload a compound library and protein file for screening. The compound library must be in SDF format.

Next Steps:

  1. Select a PDB file that contains either the entire protein or just the active site in PDB format.
  2. Choose a prediction model to estimate the binding affinity of the protein-ligand complex.

Downloads

Information Regarding the Downloaded Files:

  1. First File (SDF - Compound File): This file contains molecular structures of chemical compounds, typically small molecules or ligands, stored in Structure-Data File (SDF) format.
  2. Second File (PDB - Whole Protein): This Protein Data Bank (PDB) file represents the entire 3D structure of a protein, including all its atomic coordinates, chains, and possibly bound molecules.
  3. Third File (PDB - Active Site): This file contains only the active site region of the protein, which is the specific binding site for ligands, crucial for molecular docking and simulations.
